Usage
---

#### mixin

```jade
include mixin/all
```

#### layout

```jade
extend layout/basic
```


Layout
---

If you use layout like <layout/basic.jade>

- use `block` to replace the default value
- use `append` to insert after block
- use `prepend` to insert before block
